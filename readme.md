### F1Tenth Autonomous Racing

ROS2 implementation 

- All the packages are in ROS2 galactic

Launch file
```
ros2 launch autodrive_f1tenth simulator_bringup_headless.launch.py
```

Teleop Command
```
ros2 run autodrive_f1tenth teleop_keyboard
```
